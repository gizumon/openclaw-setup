# Claude Code ルール

## ドキュメント化ポリシー

後で使い回せるコマンドや手順が出てきたら、必ず `TIPS.md` に追記する。

### 対象となる情報

- CLI コマンド（gcloud, terraform, npm など）
- 接続手順（SSH, Port Forward, API 認証など）
- トラブルシューティングで判明した対処法
- 設定ファイルのパスや重要な値

### ルール

- **新しいコマンドや手順を使ったら `TIPS.md` に追記する**
- 既存のセクションに追加できる場合はそこへ。なければ新しいセクションを作る
- コマンドはコードブロックで記載し、変数部分は `${変数名}` 形式にする
- 1コマンドずつ何をするものか一言添える
